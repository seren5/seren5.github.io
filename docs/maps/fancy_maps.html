Fancier Maps

We can use leaflet to make some fancier maps.

#install.packages(leaflet)
library(leaflet)

Manual here Here is a text on leaflet.

leaflet()|>
  addTiles()

Let’s put Smith College on the map. Here’s the lat and long.

leaflet()|>
  addTiles() |>
  addMarkers(lng = 72.6403,
             lat = 42.3163,
             popup = "Home of the Pioneers",
             label = "Smith College")

Whooops. We need a negative sign to indicate west.

leaflet()|>
  addTiles() |>
  addMarkers(lng = -72.6403,
             lat = 42.3163,
             popup = "Home of the Pioneers",
             label = "Smith College")
leaflet()|>
  addTiles() |>
